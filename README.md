# E-Commerce Product Catalog

A modern and responsive E-Commerce Product Catalog built using HTML, CSS, and JavaScript as part of my internship project.

## Features

* Home Page
* Product Listing Page
* Product Detail Page
* Search Products
* Category Filter
* Add to Cart Functionality
* Cart Page
* Local Storage Support
* Responsive Design
* Client-Side Routing

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
